Mark Book

Source

This is a mark book program that keeps track of marks achieved in school. It computes the grade through weights of each deliverable for an accurate representation of the grade. The GUI is developed using the Tkinter Framework

UofT TimeTable Tool

Source

This is a program that was created as a part of the design project in the Fall 2021 term of CSC207H1F at the University of Toronto St.George. It is a timetable planning tool, that gathers information from the UofT Course Finder using the JSoup library. Users can search and add course or other tasks and activities. They can load and unload to a serialized file with their courses. Finally, a Depth First Search Algorithm is used to automatically schedule multiple courses into the timetable.
