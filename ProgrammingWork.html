UofT TimeTable Tool Program

The Program

This is a program that was created as a part of the design project in the Fall 2021 term of CSC207H1F at the University of Toronto St.George. It is a timetable planning tool, that gathers information from the UofT Course Finder. Users can search and add course or other tasks and activities. They can load and unload to a serialized file with their courses. Finally, a Depth First Search Algorithm is used to automatically schedule multiple courses into the timetable
